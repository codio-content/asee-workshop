## ⚡ Quick Start

If you haven't already, do these three things:

**1. Install OpenCode Desktop**

**2. Clone this repo using HTTPS**

```bash
git clone https://github.com/codio-content/asee-workshop.git
cd workshop
```

> ⚠️ Use the HTTPS URL above — not SSH. SSH requires linking your GitHub account first, which we're skipping to save time.

**3. Open this repo directory on your file system as a "Project" in OpenCode Desktop**


You're ready.
